# Welcome to your Lovable project
🌉 ConceptBridge
Bridging the gap between STEM theory and real-world application.

🔗 View Live Demo

🎯 The Mission
Students often learn "how" to solve an equation but not "why" it matters. ConceptBridge uses AI to instantly connect academic concepts (Physics, Math, Electronics) to modern industry use-cases.

✨ Key Features
Simple Logic: Explains complex topics in plain English.

Industry Use: Shows how companies like SpaceX or Tesla use these concepts.

Visual Discovery: Dynamic images for every searched concept.

Gamified Learning: Interactive quizzes and career pathway suggestions.

History: Tracks your learning journey via local storage.

🛠️ Tech Stack
Frontend: React + Tailwind CSS

AI: Gemini / OpenAI API

Build Tool: Lovable

📖 Quick Start
Type a concept (e.g., "Centripetal Force") in the search bar.

Read the "Foundation" and "Industry" cards.

Take the Quick Quiz to test your knowledge.
